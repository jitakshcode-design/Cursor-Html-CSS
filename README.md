# Cursor Landing Page UI Clone 

## Here, I am have cloned UI of Cursor landing page using HTML and CSS.

Build this basic project for better understanding of HTML and CSS. By this projects I have cleared by concept about Semantic HTML and Flexbox and Grid in CSS. 

### The Section that i have recreated from the original landing page


1. Navigation Bar - fixed it using using position property.added CTAs
2. Hero Section - Big Picture and download for Mac Section
3. Trusted by coompanies
4. Feature card section - grid of 3 cards - alternative picture and text
5. Use Cases and Story Section
6. Use Case/Stories Section - used both flexbox and grid
7. Updates - grid of 4
8. Joining team - cta,text and picture
9. final cta
10. footer

### Colors and text used -


1.Background color - #151512

2.Text Color- #F7F7F4

3.Font Family - system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;
